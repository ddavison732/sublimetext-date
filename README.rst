sublimetext-date
================

A plugin for Sublime Text 2 that inserts date and hour when f5 is pressed.

Simple, yet, i kinda like it.

Usage
~~~~~

::

	F5: Prints date (DD/MM/AAAA) followed by hour (HH:MM)
	shit+F5: prints only hour (HH:MM)
